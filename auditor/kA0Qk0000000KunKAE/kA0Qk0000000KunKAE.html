Overview

In some cases, Activity Summary emails may still have .zip files attached, even if the entire Attach Activity Summary as a CSV file tree checkbox in the monitoring plan settings is unchecked. The Instructions section provides additional information on the in-depth Activity Summary emails setup.

Instructions

IMPORTANT: Make sure to stop the corresponding collector service before introducing the changes. Start the service once the changes are saved.

Each Auditor collector has a separate NwxEmailNotificationClient.acinc file located in the corresponding C:\Program Files (x86)\Netwrix Auditor\* folder. Before editing, pay attention to the Attach Activity Summary as a CSV file state in the monitoring plan:

  • If Attach Activity Summary as a CSV file is checked, the MaxAttachmentFileSizeInKB subnode will be active.

  • If Attach Activity Summary as a CSV file is not checked, the MaxAttachmentFileSizeInKB subnode will not be active.

 

Depending on the ticket, the customer might need a different resolution. Refer to the following subnode descriptions to resolve possible issues:

  1. The <a n="MaxChangesInEmailBody" v="3000"/> subnode states the max amount of ARs that will be listed in a single email.

    • If Attach Activity Summary as a CSV file is not checked, and the actual number of ARs is higher than stated, a .csv file is generated, compressed into .zip, and attached to the email.

    • This does not affect the checked Attach Activity Summary as a CSV file option.

  2. The <a n="MaxEmailXmlSizeInKB" v="10240"/> subnode states the max size of the email before it is compressed.

    • If Attach Activity Summary as a CSV file is not checked, and the actual email XML table size is higher than stated, a .csv file is generated, compressed into .zip, and attached to the email.

    • This does not affect the checked Attach Activity Summary as a CSV file option.

  3. The <a n="MaxAttachmentFileSizeInKB" v="5120"/> subnode states the max size of the attached .csv file.

    • This does not affect the unchecked Attach Activity Summary as a CSV file option.

    • If Attach Activity Summary as a CSV file is checked, and the actual attachment size is larger than stated, the emails are split in parts. The attached .csv files are split into parts (e.g., Summary_Details - part 2 of 8.csv).
