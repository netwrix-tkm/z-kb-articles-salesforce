Symptoms

Refer to the list of possible symptoms in your Netwrix Auditor (NA) environment:

  • After you set up the Netwrix Privilege Secure for Access Management (NPS AM) integration, NA prompts the following error in the Health Log for your Active Directory monitoring plan: 

    The user name or password is incorrect

    The ADA error may be represented by a different message:

    Failed to collect Active Directory changes due to the following error:
Failed to get any valid DC.
The following errors occurred:
Failed LDAP validation of DC %FQDN% with error: The user name or password is incorrect.
Failed LDAP validation of DC %FQDN2% with error: The user name or password is incorrect.
Failed to obtain DC by locator (Netlogon service) with error: The user name or password is incorrect.

  • The errors are present in an environment with multiple DCs.

Cause

This is a known issue confirmed by the R&D team.

Resolution

Temporarily disable the NPS AM integration.
