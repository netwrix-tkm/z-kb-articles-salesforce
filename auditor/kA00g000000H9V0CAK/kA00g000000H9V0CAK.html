How to omit some Exchange Server attributes from being reported?
To omit an Exchange Server attribute from reporting:
  1. Navigate to Netwrix Auditor installation folder. For example, C:\Program Files (x86)\Netwrix Auditor\Active Directory Auditing.
  2. Locate omitproplist_ecr.txt file and add the attributes you want to exclude from reporting in the following format:
*.<attribute name>
For example: *.msExchSafeSendersHash
