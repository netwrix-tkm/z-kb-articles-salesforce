Overview

The Active Directory (ADA) collector in Netwrix Auditor supports the Signing and Sealing encryption method for LDAP. This article covers the steps to enable encryption.

Instructions

Refer to the following steps:

  1. In the Auditor server, run the following line in an elevated PowerShell instance to stop the Configuration Server Service: 

    Stop-Service -DisplayName "Netwrix Auditor Configuration Server Service"

  2. In the Configuration.xml file, locate the following node:

    <n n="CollectorsSettings"/>

  3. Replace the node with the following node:

    <n n="CollectorsSettings">
    <a n="SigningAndSealing" t="7" v="True"/>
</n>

    Save the changes.

  4. Run a registry entry containing the following lines in the Auditor server:

    Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Netwrix Auditor\AD Change Reporter]
"EnableSignAndSealing"=dword:00000001

  5. Run the following line in an elevated PowerShell instance to start the Configuration Server Service: 

    Start-Service -DisplayName "Netwrix Auditor Configuration Server Service"
