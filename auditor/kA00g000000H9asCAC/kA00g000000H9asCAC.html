You are monitoring EMC celerra NAS and sessions are being marked as completed with warnings, however reports contain neither warning details or error text.
In most cases this means that your Celerra device is not configured properly or its security log file is not accessible for Netwrix Auditor - File Servers.
1. Please run on Netwrix host machine Event Viewer and connect to your NAS device using Netwrix Auditor - File Servers data processing account credentials. 2. Check that the log is accessible. 3. Check that it is populated with new events. If one of the steps fails, please reconfigure you NAS device as per our guide
