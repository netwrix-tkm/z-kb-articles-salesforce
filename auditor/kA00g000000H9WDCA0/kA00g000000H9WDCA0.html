How to change the frequency of data collections for File Server auditing.
In order to change the frequency of data collections please perform the following:

- Go to Control Panel -> Administrative tools -> Task Scheduler -> Task Scheduler Library;
- Double click task Netwrix Management Console - File Server Change Reporter your_collection;
- Then Triggers tab, then click New and choose Daily and set Repeat task every N hours (type your selection if it is not available to pick from the drop down).
