How do I filter out certain mailboxes and folders from product reports?
To filter out certain mailboxes and folders, navigate to the program installation folder and open the mailboxestoexclude.txt file. This file contains a list of mailboxes and folders to be excluded from reports.

You can specify ‘Mailbox_Name’, ‘Mailbox_Name/Folder_Name’ or use wildcards ‘*/Folder_Name’. The latter excludes the specified folder of all mailboxes.

NOTE: If compression service is not enabled, the ‘Mailbox_Name/Folder_Name’ lines are ignored.
