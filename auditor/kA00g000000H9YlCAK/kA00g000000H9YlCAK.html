When auditing NetApp Data ONTAP, you get the following error in the Netwrix Auditor System Health log: Event 2002 Cannot collect data on the object '\server_nameshare_name.copy_offload.tokens': Error: Access is denied.
Your file share contains the system .copy_offload folder used by Microsoft Offloaded Data Transfer (ODX). Access to system files and folders inside .copy_offload is restricted.
It is recommended to exclude system files and folder from auditing. To do this, perform the following steps:
  1. Navigate to %Netwrix Auditor Installation folder%File Server Auditingomitcollectlist.txt.
  2. Add a new entry to omitlist.
For example: *,,\\\\server_name\\share01\\.copy_offload*
