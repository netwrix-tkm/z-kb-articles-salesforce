How do I omit changes to the "msExchMobileMailboxPolicyBL" attribute?
To exclude changes to the "msExchMobileMailboxPolicyBL" attribute from being reported please perform the following steps:
1. On the machine where Netwrix Auditor is installed navigate to the product installation directory. By default it is C:\Program Files (x86)\Netwrix Auditor\Active Directory Auditing
2. Open omitproplist_ecr.txt file for editing and add the following line: *.msExchMobileMailboxPolicyBL*
3. Save this file
