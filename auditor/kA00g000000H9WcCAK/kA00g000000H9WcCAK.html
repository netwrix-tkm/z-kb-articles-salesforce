How to control the size of NetwrixSQLCRAudit databases on SQL instances and what they are used for.
If "Audit data changes" option is enabled as a part of SQL Server audit, Netwrix Auditor creates an additional database - NetwrixSQLCRAudit against each monitored SQL server, the database is used as a temporary storage of all changes performed to other server databases. In order to control the size of NetwrixSQLCRAudit database the following two options are suggested:
1. Disable the "Audit data changes" option, if you are not interested in content changes and delete the NetwrixSQLCRAudit database(s) from the SQL Server(s).
