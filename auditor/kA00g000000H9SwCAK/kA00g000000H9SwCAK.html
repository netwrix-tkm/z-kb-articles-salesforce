How to exclude Smart Card users from Password Expiration Notifier monitoring scope?
Perform the following steps:
  1. On the computer where Netwrix Auditor Server resides, open Registry Editor: navigate to Start -> Run and type "regedit";
  2. In the Registry Editor, create a new registry key:
  • HKEY_LOCAL_MACHINESOFTWAREWow6432NodeNetwrix AuditorPassword Expiration Notifier<domain name>.
  1. Create the new DWORD IgnoreUsersWithSmartCardsSettings value under this key and set it to 1.
