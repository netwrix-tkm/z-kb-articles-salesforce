How do I create new reporting databases?
You can backup and delete the existing Netwrix databases with SQL Management studio or just delete them depending on your preference. You can then create a new database for each audited system by re-applying the SQL report settings for your managed object in the Netwrix Auditor console (Netwrix Auditor console | Manage Objects | Your Manage object | Audited System (for example Active Directory) | Reports | Report settings tab on the right | Apply button).
