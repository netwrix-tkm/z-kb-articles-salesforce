Symptom

The following error is prompted in Health Log for your Microsoft 365-based monitoring plan: 

AADSTS700027: The certificate with identifier used to sign the client assertion is not registered on application. 
Reason - The key was not found. Thumbprint of key used by client: '%thumbprint%'

Cause

Multiple Microsoft 365-based items use the same Entra ID application instead of dedicated applications. This causes the certificate thumbprint overwrites.

Resolution

Set up a separate Entra ID application for every Microsoft 365-based item monitored in your environment. Refer to the following articles for additional information on Microsoft 365-based sources: Microsoft 365 — Permissions for Entra ID Auditing ⸱ v10.6, Microsoft 365 — Permissions for Exchange Online Auditing ⸱ v10.6, Microsoft 365 — Permissions for SharePoint Online Auditing ⸱ v10.6, and Microsoft 365 — Permissions for Teams Auditing ⸱ v10.6.
