Question

Your event log contains a login by a Font Driver Host\UMFD-% user. What does a UMFD-% user represent?

Answer

The User-Mode Font Driver Host service (UMFD) being a part of User-Mode Driver Framework (UMDF) is an integral part of Windows OS. It is a system process caching a font driver. To get additional information on the logon, refer to the following steps:

  1. Collect date and time of the logon.

  2. In Search, use the Where filter to specify the DC the user logged on, and check the corresponding Security event log.
