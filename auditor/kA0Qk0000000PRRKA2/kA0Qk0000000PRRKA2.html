Symptoms

  • The data collecting account in the UAVR monitoring plan regularly gets locked out.

  • The password specified in monitoring plan settings is confirmed to be correct.

Cause

The password is not propagated in the Agents.xml file.

Resolution

  1. Stop Netwrix Auditor User Activity Audit Service.

  2. Back up the contents of the %Working Folder%\User Activity Video Reporter directory, and delete the contents. Alternatively, EncryptedPassword entries in the Agents.xml file can be deleted to allow Auditor to propagate the actual password automatically.

  3. Start Netwrix Auditor User Activity Audit Service.
