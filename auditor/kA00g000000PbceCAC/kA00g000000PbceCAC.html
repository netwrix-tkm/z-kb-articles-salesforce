1. In administration web console, navigate to Config -> Settings -> Core.
  1. Click the Run Cleaner button in the right corner.
  2. In the Index Maintenance dialog, select Re-Collect Index.
  3. On the Would you like to re-run the product configuration wizard? step, select Run and click Next.
  4. On the Summary step, click Next.
  5. Click Yes in the confirmation dialog that appears.
  6. Wait until all actions complete.
  7. Once completed, the Initial Product Configuration wizard appears.
  8. On the Instance step, select Instance that you want to proceed with.
  9. Click Next.
  10. On the Processing Settings step, select desirable Index Processing mode: No Index, Keyword or Compound Term.
