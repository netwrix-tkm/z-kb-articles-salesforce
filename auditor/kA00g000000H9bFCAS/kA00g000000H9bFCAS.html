You are getting the following warning message on the 'Exchange Server Administrator Audit Logging Settings' step in the Audit Configuration Wizard: Exchange Server: <Exchange server name> Unable to establish connection to the following Exchange server: '<Exchange server name>'. The following error occurred: Remote Session is not available for <Exchange server name>..
Typically this is a permission issue for the account under which Audit Configuration Wizard is opened.
Make sure that the user running the Audit Configuration Wizard is added to the Organization Management security group.
