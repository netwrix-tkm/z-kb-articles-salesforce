Netwrix Password Expiration Notifier sends emails to AD users when their passwords are about to expire. How can this be configured for the service accounts that run server-based applications? How can the IT staff be notified of an impending service account password expiration?
In Netwrix Password Expiration Notifier, select the Send report to the users' managers check-box. Using the Active Directory Users and Computers tool, specify the manager's email address for a service account. The manager will be notified about the service account password expiration.
