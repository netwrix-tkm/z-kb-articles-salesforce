You received a Change Summary or a report containing the following warning message: "Warning: Collect 'Database' information warning on server: server_name DB: db_name. Database ' db_name ' is already open and can only have one user at a time".
Access to the database is restricted to one user at a time. Netwrix Auditor failed to access the database because another user was working with it.
To change the database access restriction, perform the following steps:
  1. Open Microsoft SQL Server Management Studio and connect to your server.
  2. Navigate to the required database, right-click it and select Properties.
  3. Open the Options tab and change the "Restrict Access" value to "MULTI_USER".
