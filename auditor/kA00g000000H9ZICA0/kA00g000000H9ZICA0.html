You get the following error after opening the Netwrix Management Console:
This issue happens when the console verifies the difference between local date on the machine and the registry key value.
To fix this issue please make sure that the local date on the machine where the Netwrix Management Console is installed has the same format as found in the registry key - InstallDate . To find it please open the registry (Start | Run | regedit) and navigate to: - For Windows x64 : HKEY_LOCAL_MACHINESOFTWAREWow6432NodeNetWrixUser Activity Video Reporter - For Windows x86 : HKEY_LOCAL_MACHINESOFTWARENetWrixUser Activity Video Reporter
