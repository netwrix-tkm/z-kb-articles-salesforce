Email reports contain the following warning: "Some required components are not installed on the target computer" .
NetWrix Server Configuration Change Reporter uses the WMI Windows Installer Provider to monitor changes.
To be able to collect and analyze changes, perform the following steps:
  1. Logon to the server specified in the warning message. Open Control Panel / Add or Remove Programs/ Add/Remove Windows Components
  2. Double-click Management and Monitoring Tools. Select WMI Windows Installer Provider and click OK. Click Next to start the installation.
  3. If the same warning appears for the same machine in the next report, contact NetWrix Technical Support.
