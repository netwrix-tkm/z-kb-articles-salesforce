How to count number of CPU cores on your Oracle Database deployment?
The licensing for Netwrix Auditor for Oracle Database is based on the number of CPU cores utilized by the entire Oracle Database deployment (the CPU_CORE_COUNT_CURRENT parameter). To find the overall core count, you will need to retrieve the number of CPU cores on each Oracle Database instance and put the numbers together. Perform the following steps:
  1. On the computer where your database is deployed, run the sqlplus tool.
  2. Connect to your Oracle Database instanceâ€”use Oracle account with the SYSDBA privilege. For example:
OracleUser as sysdba
Enter your password.
  1. Execute the following SQL query:
select CPU_CORE_COUNT_CURRENT from V$LICENSE;
Repeat these steps for every Oracle Database instance and then sum up the results to receive the number of CPU cores for the entire deployment.
For details on the Oracle Database commands mentioned in this article, refer to
