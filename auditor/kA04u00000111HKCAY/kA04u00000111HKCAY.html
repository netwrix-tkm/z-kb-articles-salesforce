Symptom

After a user changes their password as per the password policy enforced in the environment, Netwrix Password Expiration Notifier still sends emails to the affected user to prompt the password change.

Cause

A rare version-specific issue present in 10.5.10977.

Resolution

Upgrade the Netwrix Auditor instance to the latest version available. A fix has been confirmed for the upgrade from 10.5.10977 to 10.5.11059.
