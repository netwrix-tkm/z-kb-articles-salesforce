Problem


When deploying the app to a SharePoint site collection it fails. The error written to the conceptsearching log on the NDC server is: 
Error: When deploying to a 'noscript' site collection please use a tenancy scoped deployment.

Resolution:  

  1. Download & Install SharePoint Online Management Shell
  2. Open SharePoint Online Management Shell
  3. Connect to your SharePoint Admin Portal
  4. Execute the following PowerShell command: 
    Set-SPOsite https://*your-domain.sharepoint.com*/site-path -DenyAddAndCustomizePages 0
  5. Retry the app deployment.
