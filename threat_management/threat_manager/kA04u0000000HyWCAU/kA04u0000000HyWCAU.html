The following table contains a list of the variables available for use in the StealthDEFEND SIEM template:

VariableDescription
%SYSLOG_DATE%DateTime.UtcNow
%HOST%StealthDEFEND Server Hostname
%COMPANY%"Stealthbits"
%PRODUCT%"StealthDEFEND"
%PRODUCT_VERSION%StealthDEFEND Version
%THREAT_TIME%Threat Detected Time
%THREAT_TYPE%Threat Type
%USERS%Threat Perpetrator
%COMPUTERS%Threat Host (Typically Domain Controller or File Server)
%FILENAME%File or Share Name for file events
%NEW_FILENAME%New File Name (for rename events)
%PROCESS%Process Name
%EVIDENCE%Threat Evidence (Summary/Supporting information for Threat)
