Summary: Enabling additional logging for the StealthINTERCEPT Admin Console

Issue: How to enable additional logging for the StealthINTERCEPT Admin Console.

Instructions:

There are two options for enabling additional logging: 

Option one: 
Step 1 - Access the SI management server.
Step 2 - Open the Enterprise Manager installation folder.  The default location is ...\STEALTHbits\StealthINTERCEPT\SIWinConsole.
Step 3 - Edit the file called SIWinConsole.log.config in your favorite text editor.
Step 4 - Change WARN to DEBUG in the following portion of that file:

 
   
   
   
 

Step 5 - Save the config file and restart the StealthINTERCEPT Admin Console.


Option two: 
Step 1 - Open StealthINTERCEPT Admin Console on the SI management server.
Step 2 - Select 'Agents'
Step 3 - Click on the Pencil icon at the top left of the window.
Step 4 - Change the drop-down under Administration Console  to DEBUG.
Step 5 - Click the Update Log Levels button.

After collecting debug logs switch StealthINTERCEPT Admin Console logging back to WARN.
