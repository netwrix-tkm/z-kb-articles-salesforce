Summary:
104957: MS updates: SUVP 22-06 support
104299: authentication lockdown policy not limited to specified Active Directory perpetrator.
100143: LDAP shared Memory IPC overflow
Behavior before Fix:
104957: SI agents may lose ability to distinguish Local logins to DC. Therefore, filtering by 'Login Type' for DCs may work incorrectly for Authentication policies
104299: Authentication lockdown policy does not block the managed service account users
100143: A large flow of LDAP events leads to 'shared Memory IPC overflow' error in the Agent and some event can't be sent to the Agent from AD monitor.
Behavior after Fix:
104957: Signatures updated
104299: Authentication lockdown policy block the managed service account users
100143: New filter is implemented in the AD monitor and helps reduce the number of events to send to the agent.
