Title:

activitymonitor-7.1.252

Summary:

  1. Large Downloads from Office.com
  2. Large number Nasuni events being captured with 'UNKNOWN' protocols and different paths

Behavior Before Fix:

  1. When the number of EXO DLP events exceeded the page size limit used by NAM for queries, NAM failed to correctly save a bookmark to the last read DLP event. This resulted in NAM repeatedly querying the same data pages.
  2.  Unknown events are being reported for Nasuni.

Behavior After Fix:

  1. Implemented fix when page size limit is reached.
  2. Unknown Nasuni events are now filtered out.
