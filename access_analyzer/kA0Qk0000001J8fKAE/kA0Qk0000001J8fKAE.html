Symptom

Netwrix Enterprise Auditor prompts the following error:

Error enumerating remote shares: Access is denied

Cause

The service account has insufficient rights to perform the scan.

Resolution

Assign the sufficient right to the service account. Depending on the target operating system version, refer to the following required rights:

  • When you target a server running Windows Server 2022 and later, verify that the service account is a member of one of the following groups:

    • Administrator

    • Access Control Assistance Operators

    • Server Operator

  • When you target a server running Windows Server 2019 and earlier, verify that the service account is a member of one of the following groups:

    • Administrator

    • Power User

    • Print Operator

    • Server Operator
