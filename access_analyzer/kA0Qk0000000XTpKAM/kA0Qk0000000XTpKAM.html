Symptom

Netwrix Enterprise Auditor prompts the following errors:

Failed to create FSAdapter (80070002): The system cannot find the file specified
Error during processing: Failed to create FSAdapter

Cause

Your installed versions of Enterprise Auditor and the Sensitive Data Discovery Add-On are incompatible.

  • Enterprise Auditor v11.6.0.59 and higher require Sensitive Data Discovery Add-On v11.6.0.10 and higher.

  • The Sensitive Data Discovery Add-On v11.6.0.10 and higher require Enterprise Auditor v11.6.0.59 and higher.

The incompatibility is caused by an outdated component.

Resolution

Update both Enterprise Auditor and the Sensitive Data Discovery Add-On to their latest versions. Refer to the following articles for additional information: Sensitive Data Discovery Add-On Installation — Upgrade Sensitive Data Discovery Add-on · v11.6 and Installation & Configuration Overview − Enterprise Auditor Console Upgrade · v11.6.
