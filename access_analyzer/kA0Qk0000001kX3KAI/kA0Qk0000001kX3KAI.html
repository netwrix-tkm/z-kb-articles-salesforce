Symptom

When navigating through the NEA console by selecting jobs or job groups to display the job information on the right side of the console in Netwrix Enterprise Auditor (NEA) version 11.6, the system generates the following error message: 

Data at the root level is invalid. Line 3, position 5

Cause

These errors may be caused by any one of the following:

  • The Reports folder at this location: \%SAInstallDIR%\StealthAudit\Reports> is missing the Reports.xml file.

  • The Reports.xml file is corrupt.

Resolution

To address these causes, remove the corrupt Reports.xml file and reopen the NEA application to generate a new Reports.xml file.
