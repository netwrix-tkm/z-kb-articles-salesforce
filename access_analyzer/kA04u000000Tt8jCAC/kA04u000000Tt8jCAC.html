Summary:
  1. SAM is now reporting UNC Paths for nasuni devices
  2. Netapp showing read for all subfolders
  3. SAM Linux agent showing outdated in console
Behavior Before Fix:
  1. There was a bug reporting UNC Paths for nasuni devices.
  2. For Netapp SAM reported extra read events of subfolders caused by behavior of file explorer.
  3. Issue deploying SAM Linux agent using certificate only.
Behavior After Fix:
  1. UNC Paths are not reported correctly for nasuni devices.
  2. Extra subfolder read events are not reported for netapp.
  3. Deploying SAM Linux agent using certificate only works as expected.
