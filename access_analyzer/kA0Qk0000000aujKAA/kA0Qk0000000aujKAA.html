Question

What can be done to address the issue of getting a pop-up that the script is not responsive and experiencing console freezing?

Answer

A: If you encounter a pop-up indicating that the script is not responsive or experience console freezing in Netwrix Enterprise Auditor, follow the steps below to resolve the issue:

  1. First, attempt to clear out the job statistics table to see if the issue is resolved. If you have already tried this step and it did not solve the issue, proceed to the next steps.

  2. Run the following SQL code to create a non-clustered index on the SA_Messages table: 

    CREATE NONCLUSTERED INDEX [Messages_JobName_JobRunTimeKey_Index] ON [dbo].[SA_Messages]
(
[JobName] ASC,
[JobRunTimeKey] ASC
)
INCLUDE([MSG_TYPE])

  3. After creating the index, monitor the system to observe if the script responsiveness improves and console freezing issues are mitigated.
