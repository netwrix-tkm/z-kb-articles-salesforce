Summary:
  1. Error found that seems to clear itself up - Unsure if its affecting anything  - Failed to create Scope message.
  2. MS updates: SUVP 23-04 support - no NTLM events
Behavior Before Fix:
  1. For Netapp, SAM didn't check if scope already existed before creation of scope. (New versions of netapp started to report errors for this.)
  2. SI Agent does not capture NTLM events due to signature changes in NlpLogonSamLogon.
Behavior After Fix:
  1. For Netapp,  SAM now checks if scope exists and modifies it as needed.
  2. Signatures updated; NTLM must be fully functional.
