Notes:
This will not recover information that was deleted until a full scan, but it will fix for scans going forward.
Also updated the SA_AzureADInventory_GroupMembersView, SA_AzureADInventory_EffectiveGroupMembersView, SA_AzureADInventory_GroupOwnersView and effective group members calculation to consider users and groups that were flagged as Deleted in the database.
The Graph Groups delta API does not inform you that a member was removed from a group if you simply delete the User object in Microsoft Entra ID, so this will now reflect more accurate group member information.
This escalation will also include 2 HF’s this is the first one.

Instructions:
  1. Unblock the hotfix zip file in the windows property dialog, if an unblock button exists there. 
  2. Close all instances of StealthAUDIT (check task manager under processes for all users)
  3. Update all DC files in your %SAInstallDir%DC folder
  4. Update all PrivateAssemblies in your %SAInstallDir%PrivateAssemblies folder.
  5. Launch SA
