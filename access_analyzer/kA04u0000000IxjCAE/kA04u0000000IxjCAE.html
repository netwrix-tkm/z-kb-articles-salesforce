Summary: StealthAUDIT (SA) published reports with the option to download to csv opens directly in default browser instead of as a download

Issue: SA published reports can be configured with the option to download to csv (AllData). When clicking, this opens the CSV file directly in the default browser rather than prompting to save. This happens when windows has no application associated with the file extension ".csv".

Instructions: :  To resolve the issue, use the below instructions to associate the file with notepad, which are Windows O.S specific:
 
Server 2016\2019:

  • Windows Settings\Apps, choose default apps by file type
  • Scroll down to file extension .csv, select choose default app.  Select Notepad

Server 2012/R2:

  • Control Panel\Programs\Default Programs
  • Select Associate a file type or Protocol with a specific application
  • Scroll down to .csv select Change program. Select notepad

Server 2008:

  • Control Panel > Default Programs.
  • Select CSV and change program, select Notepad

 
Windows Desktop OS:

  • If Microsoft Office is installed, CSV files are likely already associated with Excel will not need to do this.

 Files may be associated with your choice of program, not just Notepad.
