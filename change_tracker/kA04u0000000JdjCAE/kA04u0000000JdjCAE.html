This is useful when you are receiving several change events in process output that you want to exclude. Examples of unwanted information that will always change include "uptime" and "Time Source" information.

Step-by-step guide

Please follow the steps below to Exclude/Edit Information from Process Output Tracking:

  1. First go to "Settings"
  2. On the sidebar, click "Credentials"
  3. Select the credentials that fall under this device and click "Edit"
  4. See Screenshots Below:
  5. Next, we will need to edit the login script to exclude this line from appearing in the .
  6. Please enter the text below above everything in the login script:

    ExcludeMatchesWithComment,**Variable One**,Variable Two
     
  7. Variable One: Edit this text andEnter what you would like the edited message to be.
  8. Variable Two:Edit this text andEnter the first few words in the line you wish to exclude/edit
  9. See Screenshot below for Example:

     
  10. After adding this line, click update and start the process output tracker from the devices screen
