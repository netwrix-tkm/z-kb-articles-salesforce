Step-by-step guide

This error is usually caused due to the process tracker in change tracker being enabled, but no specified processes are set to be tracked. To stop this error from occuring, you will need to edit the configuration template assigned to the device that the error is occuring on, and either input at least one process to track, or disable the tracker

  1. Log Into Change Tracker and Navigate to Devices > Select Device Experiencing Error > Select the Configuration Template that is causing the issue
  2. Either Uncheck the "Track processes and services in this template" box, or add some services to be tracked. See example below:
