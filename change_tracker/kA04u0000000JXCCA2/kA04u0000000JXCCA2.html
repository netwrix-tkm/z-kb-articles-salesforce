This is only applicable for Windows Server 2012R2 or Later versions - Earlier versions such as Server 2008R2 will not be able to run this script.


Below you will find the link to the powershell script that will install the prerequisite roles and features. If you cannot download the powershell scrit - a .TXT file has been attached that can be renamed  


Powershell Script - PS1: Gen7-Roles-and-Features-Install-Script-Server-2012-and-Later.ps1
Powershell Script - TXT: Gen7-Roles-and-Features-Install-Script-Server-2012-and-Later.txt


In the case that you have manually installed the roles and features - the script can be run to make sure that nothing was missed. If something was missed during the manual process, it will install it; otherwise it will skip over it.

Here is to use the script:

If you have decided to go with the TXT version - Please rename this from Gen7-Roles-and-Features-Install-Script-Server-2012-and-Later.txt TO Gen7-Roles-and-Features-Install-Script-Server-2012-and-Later.ps1


  1. open powershell as administrator, and navigate to the script directory using a "cd" command - For the downloads folder, an example is below: 

              a.  cd "C:\Users\My Username\Downloads"

       2. Issue the following Command:

              a.  ./Gen7-Roles-and-Features-Install-Script-Server-2012-and-Later.ps1


Note: you may need to issue a "Set-ExecutionPolicy Unrestricted" command prior to running the powershell script. Once complete, to reset this to default, please run a "Set-ExecutionPolicy Restricted" command.


Also: You may also need to allow powershell to run an unsigned script if you have downloaded the ".ps1" - You can go ahead and type in "R" and hit enter.
