Users: User Administration
Edit User attributes such as username, assigned system privileges, email address and password.
Adding a new user requires a Time Zone to be selected in order that the Dashboard screen correctly shows the relevant time periods for the user
User Roles that are pre-defined include Analyst/User (essentially Read Only access), User Administrator (Read Only access plus basic Admin rights), Agent (Agent Only access – Agent registration and polling access only) and Administrator (Full Admin rights). Other Roles can be created – please contact NNT for more information
By checking the AD Login box for any user, the user login authentication for CT will be provided by the network's AD or LDAP server – this will need to be defined separately under Settings -> System Settings
You can restrict a User's access to Device Groups by editing the Restricted to Groups field – by default this is blank and provides the user with full visibility of all Groups.
Password is set/changed by clicking the chain icon
