Stop the IIS Web server using command line iisreset /stop or use the IIS Management Console, or use the Windows Services Console.

Next, stop the Mongo DB service using the Windows Services Console.

Edit the mongod.conf file located in C:\Program Files\NNT Change Tracker Suite\Gen7\MongoDB\conf

Edit the parameters for dbpath to change the location for DB files.

Then start the Mongod service, followed by an iisreset /start.
