Step-by-step guide

This error occurs to your account not having the correct user rights to view the groups page. We have resolved this in previous scenarios by doing the following:

  1. Log out of the change tracker user account that you are currently logged in as
  2. Log in as the default "admin" account in Change Tracker.
  3. Navigate to the "Groups" tab in the Settings tile.
  4. Do you see the groups? You should be able to. This means that the account you previously had logged in as did not have the user rights to view the device groups.
  5. Go to Settings > Users
