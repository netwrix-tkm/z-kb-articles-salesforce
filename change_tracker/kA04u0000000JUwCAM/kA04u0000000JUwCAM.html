Alerting - Being Notified of Events

Once devices have been assigned to Device Groups and a Configuration Template has been assigned, a Baseline of the configuration will be recorded and thereafter, any changes detected can be reported as email or syslog alerts.

Each user of the Change Tracker system can be assigned a personalized alerting profile from a range of event options for each Device Group.

Access the User Notification Email page via Settings -> User Notification Emails and select a user, then dial in Notification Types for the selected Device Group.

Note: Core system settings such as the Syslog Server and SMTP details will also need to be defined via Settings -> System Settings in the Change Tracker Gen 7 Admin Guide for more details..


For a more detailed view of the the Saved Query and Email alert, see the Admin Guide section, Gen 7 Settings Page - User Notification Emails and Gen 7 Settings Page - Saved Queries
