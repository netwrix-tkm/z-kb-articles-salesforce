This article for when you need to export events to a file, but you have more than 50k events listed and the export button is greyed out.

Step-by-step guide

  1. First Go to "Settings"
  2. Scroll Down and click "Export and Import"
  3. For Export Type, Select "Events Only"
  4. For Export Format, Select "Text (CSV)"
  5. Select Start and End Dates to export.
  6. Then Click "Perform Export"
  7. See Screenshot Below for Example:
  8. Scroll down and download the export and unzip the CSV file.


NOTE | In some cases, you may need to do the following:


Open a powershell prompt with admin privileges and change directory to the folder where the csv is located.

Run the following command:
>Get-Content .\events.csv | Select-String 'DeviceEvent' | Set-Content DeviceEventsOnly.csv
