Question

I try to perform a MongoDump on me Mongo Database. How to troubleshoot if the MongoDump failed?

Error Message Example:

Failed: error reading collection: WSARecv tcp 127.0.0.1:50290: An operation on a socket could not be performed because the system lacked sufficient buffer space or because a queue was full.

Answer

The error message is linked to a Windows issue, rather than a MongoDB issue. The error is seen when a system tries to connect to a TCP port which is greater than 5000 and if you look at the error message the TCP port should be referenced.
Read more about cause and resolution of this issue. If possible, please implement the steps they recommend and then re-run the mongo dump task.
