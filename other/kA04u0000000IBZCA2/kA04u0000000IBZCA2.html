StealthAUDIT allows users to customize the database columns used by its analysis jobs. This article explains what each of the StealthAUDIT data types map to within the StealthAUDIT MSSQL database. 

 

BOOLEAN

SQL Date Type: varchar(5)

Value Range: “True”, “False”, or null.

 

DATETIME

SQL Date Type: datetime()

Value Range: 1753/01/01 00:00:00.000 to 9999/12/31 23:59:59.997

 

INTEGER

SQL Date Type: int()

Value Range: -2,147,483,648 to 2,147,483,647

 

NUMERIC

SQL Date Type: float()

Value Range: - 1.79E+308 to 1.79E+308

 

STRING

SQL Date Type: varchar(n)

Value Range: Up to 8,000 non-Unicode characters. Setting the size of the field to 7999 will result in an varchar(MAX) column being created, which will allow it to store up to 2GB of non-Unicode characters.

 

UNICODESTRING

SQL Data Type: nvarchar(n)

Value Range: Up to 4,000 Unicode characters. Setting the size of the field to 7999 will result in an nvarchar(MAX) column being created, which will allow it to store up to 2GB of Unicode characters.
