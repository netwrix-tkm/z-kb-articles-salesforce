Summary: TEXTSEARCH DC not enumerating subfolders properly for long file paths

Issue: Summary:  TEXTSEARCH DC not enumerating subfolders properly for long file paths
Before Fix: When scanning file paths longer than 256 characters, the TextSearch DC will throw an error saying it "Failed to GetAllSubfolders for."
After fix: TextSearch DC can scan paths longer than 256 characters without issue.
