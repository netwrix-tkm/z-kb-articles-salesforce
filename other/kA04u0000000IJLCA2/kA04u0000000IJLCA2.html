Issue: Summary:
This Hotfix addresses an issue with the job services being unable to properly process a large backlog of events, and a problem with the Action Service being unable to communicate when a self-signed certification is in use for the StealthDEFEND web service.
Behavior before Fix:

  • Large Backlogs of Events could enter a state, whereas the job service would not process the queue.
  • Action Service could have trouble connecting to StealthDEFEND when self-signed certificates are in use.
Behavior after Fix:
  • More efficient backlog processing by the Job service should resolve any issues relating to large/stuck event backlogs
  • Action Service Configuration can now optionally "ignore" potentially warnings that are a side effect of using Self Signed Certificates.
