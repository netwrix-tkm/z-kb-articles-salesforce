Issue: Summary: Customers can now add a disclaimer message that will appear in the webserver logon page
Behavior Before Fix:  there wasn't any options to add Disclaimer Message on the Report WebPage.
Behavior After Fix:  The functionality added that allows us to add the disclaimer message on Web\Disclaimer.txt, the message will be displayed on the Report webPortal

Instructions:
1. Unblock the hotfix zip file in the windows property dialog if an unblock button exists there.
2. Close all instances of StealthAUDIT (check task manager under processes for all users)

3. Exit StealthAUDIT console

4. From an elevated command prompt, navigate to the StealthAUDIT directory (default is C:\Program Files (x86)\STEALTHbits\StealthAUDIT) and run UnregisterAll.bat.

5. Stop the 'StealthAUDITWeb' services.

6. Copy and replace files from the folders in this hotfix to the corresponding folders in SA Install dir.

7. Start the 'StealthAUDITWeb' services.

8. Set the "ShowDisclaimerMessage" tag to 'True' in the \Web\WebServer.exe.config.

9. Add the Web\Disclaimer.txt file with the Custom Disclaimer Message that will appear in the webserver logon page.

10. From an elevated command prompt, navigate to the StealthAUDIT directory (default is C:\Program Files (x86)\STEALTHbits\StealthAUDIT) and run RegisterAll.bat.
