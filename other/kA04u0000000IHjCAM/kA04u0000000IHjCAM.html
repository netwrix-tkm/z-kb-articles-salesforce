Issue: Summary: SQL Connection Error - SI 6.0 Maint
Behavior before Fix:
DB Maintenance can be configured to use 'delete' or 'move' operation. For 'move' operation it is necessary to configure Archive DB settings. The code implemented to validate values entered in 'Archive DB' tab performes validation even if there are no items with 'move' operation selected.
Behavior after Fix:
SiWinConsole code updated to validate values in 'Archive DB' tab only if there is at least one item with 'move' operation selected.
