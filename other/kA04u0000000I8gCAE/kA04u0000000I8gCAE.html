Summary:

I-7125 "Track 'ntds.dit' location" feature does not work on another DC with non-default location of the file.

Behavior before Fix:

A file lockdown policy supports only default location 'c:\windows\ntds' for the 'ntds.dit' file

Behavior after Fix:

A file lockdown policy can operate any custom location for the 'ntds.dit' file
