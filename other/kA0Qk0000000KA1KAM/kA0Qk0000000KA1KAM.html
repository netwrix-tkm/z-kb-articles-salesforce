Title:
NAM_7.0_003
Summary:
  1. MS updates: SUVP 23-12 support
  2. Previously provided Home Directory fix also does not work
Behavior Before Fix:
  1. Kerberos events will be lost or incomplete.
  2. During linux agent install, if some unrequired steps fail, the total agent install would fail; If generation of client certificate failed during install, monitoring would not work after install.
Behavior After Fix:
  1. Signatures updated; SI Agents must be fully functional.
  2. During linux agent install, if some unrequired steps fail, the agent install will continue; If generation of client certificate failed during install, we now detect missing client certificate during agent refresh and generate missing client certificate.
