Summary:

No LDAP Monitor events captured

Behavior before Fix:

To define filters for LDAP monitoring SI Agent uses INI file. Syntax of INI file is case-insensitive, but the code, which finds entries in INI files, used the method of conversion to lower case characters, which failed under Turkish regional settings. As a result, SI Agent failed to find needed entries in the INI file.

Behavior after Fix:

Now code is modified in the way, which does not use conversion to the lower case to compare strings.
