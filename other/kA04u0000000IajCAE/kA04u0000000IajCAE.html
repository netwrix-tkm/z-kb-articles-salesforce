Issue: Summary: SI EM is unable to open ports for .NET Remoting on occasion
Behavior before Fix:
If port 9002 and/or port 3740 is not available for Enterprise Manager, then it will fail to start listening of this port but will continue to initialize itself. As a result, EM service will start but it will not function.
Behavior after Fix:
Now, if EM can't start listening on these ports it will retry with 5 seconds interval and will stay in 'Starting' state before it will successfully open these ports.
