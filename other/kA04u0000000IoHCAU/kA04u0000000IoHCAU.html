Summary: Exchange - Add support for 19-09 .Net Updates and KB4515832 Exchange Security Update

Issue: Summary:
Exchange -  Add support for 19-09 .Net Updates and KB4515832 Exchange Security Update
Behavior before Fix:
Exchange monitoring and lockdown policies stop working after applying one of the following Microsoft update:

  •  19-09 .Net updates;
  • KB4515832 Exchange Security Update.
Full list of 19-09 .Net KBs: https://portal.msrc.microsoft.com/en-US/security-guidance/advisory/CVE-2019-1142
KB4515832 was published 9/9/2019 for the following Exchange versions:
  • Exchange Server 2019 Cumulative Update 2
  •  Exchange Server 2019 Cumulative Update 1
  •  Exchange Server 2016 Cumulative Update 13
  • Exchange Server 2016 Cumulative Update 12
Behavior after Fix:
Extend signature file. Exchange monitoring and lockdown policies work as expected.
