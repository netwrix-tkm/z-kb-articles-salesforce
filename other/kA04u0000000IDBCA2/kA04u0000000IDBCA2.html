Summary:

Improved Navigation Menu Layout

This release features improved navigation menu for better organization of product features.

Session Extension Option

Users can now extend sessions within current RDP, SSH or Web sessions through onscreen controls. Extension limits can be controlled through Connection Profile controls.

DirectConnect RDP inline password prompt

When launching RDP DirectConnect sessions, there is no longer a requirement to alter local or group policy settings to be prompted for the user password. A password prompt is displayed in-line with the RDP session if it is not already set externally.

PowerShell Cmdlet

A PowerShell cmdlet allows common session-related functions to be scripted.

SIEM Integration

SbPAM now supports integration to any SIEM through simple and easy to use connector templates.

Domain Controller selection improvements

Better support for multi-domain controller / multi-site environments with an improved domain controller selection algorithm.

New My Activities Page

Intuitive interface presents Activities as cards for fast access to sessions.

Product Tour on First Use

When logging on for the first time, users and SbPAM admins get a self-guided product tour. Bug fixes detailed below
