3.x & Newer
Uses windows authentication and stores security descriptors for who can login in the dbo.Folder table of the NvMonitorConfig database.  
To re-gain control over SI, you can nullify securityDescriptor-s in that table.  

Step 1: Run this command against the NvMonitorConfig database:

update dbo.Folder
set securityDescriptor = null
