Summary: What is the product behavior when the license is expired for StealthAUDIT (SA)/Stealthbits Activity Monitor (SAM)? Is data sill collected when the license is expired for SA/SAM?

Issue: Is data sill collected when the license is expired for SA/SAM?

Instructions: StealthAUDIT 8.1+: When the license is expired, the scheduled jobs no longer run.  The interface will also not open, simply giving an error that the license is expired and a new one should be obtained.   Thus, even when a new license is applied, there will be no data from the time that the license was expired (but see below for SAM interaction).
 
Stealthbits Activity Monitor 3.0+:  File activity continues in SAM when the license is expired.  SAM will give a message that the license is expired, but querying for activity data during the expired timeframe will show that the program is still collecting activity.  If StealthAUDIT license is updated, data collected by SAM should be imported as normal once SA job to do so runs.

 

Product: StealthAUDIT
Module: File Activity Monitor - EMC/Celerra;File Activity Monitor - NetApp;File Activity Monitor - Windows;SA - Core
