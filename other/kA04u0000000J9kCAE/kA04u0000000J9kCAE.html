Instructions:
1. Unblock the hotfix zip file in the windows property dialog if an unblock button exists there.
2. Close all instances of StealthAUDIT (check task manager under processes for all users)
3. Exit StealthAUDIT console
4. From an elevated command prompt, navigate to the StealthAUDIT directory (default is C:\Program Files (x86)\STEALTHbits\StealthAUDIT) and run UnregisterAll.bat.
5. Copy and replace files from the folders in this hotfix to the corresponding folders in SA Install dir.
