Summary: Agent Hardening and MAC Address Resolution disabled Deployment

Issue: Summary:
SI-3941 38141 - Agent Hardening and MAC Address Resolution disabled Deployment
Behavior before Fix:
SI Agent Installer already supports modifying several parameters in the CONFIGPARAMS. The only restriction - it can modify ONLY parameters which already present in the SIWindowsAgent.exe.config file. Therefore it will not change "enableMacResolution" because it is commented out.
Behavior after Fix:
Now "enableMacResolution" is added to config file by default. This allows to modify its value with CONFIGPARAMS command line option of SI Agent Installer
Example of command line:
"SI Agent.exe" /q ENTMGR_IPADDRESS=127.0.0.1 AD_MONITOR_INSTALL=TRUE FILE_MONITOR_INSTALL=TRUE SAFEMODE=TRUE ADDFWRULES=TRUE CONFIGPARAMS="enableMacResolution=false&hardeningEnabled=true"
