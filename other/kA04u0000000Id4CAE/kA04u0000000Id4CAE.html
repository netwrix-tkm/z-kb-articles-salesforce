Summary: WinConsole License dialog doesn't show Company Name

Issue: Summary:  WinConsole License dialog doesn't show Company Name
Before fix: Information about current License is retrieved By WinConsole from Enterprise Manager and passes as LicenseData objects. There was a bug in the implementation in object de-serialization part: CustomerInfo field was de-serialized incorrectly.
After fix: LicenseData class was modified to try de-serialize CustomerInfo field in any case.
