Summary: NIS DC Unique Key Constraint: When scanning NIS environments it is possible that there can be 2 users or groups with the same exact name but with different cases (Ex: Mike and mike). SQL doesn't see these 2 values as different so when attempting to import the data it would throw a Unique Key constraint violation. The UQ constraint has been removed.
