Summary: Send Secondary Owners Email (as an option) - Issue/Current behavior: When multiple owners are assigned to a resource and a user makes a request for self-service, only the primary reviewer receives the email request. - Fix/Enhancement: A new parameter "NotifyAllOwners" was added to the common.config file, when set to 'True' all owners will be notified of access requests (self-service) and pending reviews.

Instructions:

  1. Uninstall current version of AIC if it is already installed
  2. Install version of AIC included with this hotfix.
  3. Open the C:\inetpub\wwwroot\StealthAudit Compliance\Bin\common.config file and set the "NotifyAllOwners" parameter to 'True'
  4. Log in as a requesting user and use self-service to request access to a resource that has multiple owners assigned. All the owners should be notified of the request.
  5. If a review is created on the above resource, then all of the owners should receive notification of it.
