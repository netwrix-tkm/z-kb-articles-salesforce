Summary: Not all versions and editions of SQL Server support the same level of activity monitoring. This KB provides a breakdown from SQL 2008 through 2017

Issue: Not all versions and editions of SQL Server support the same level of activity monitoring. It's important to be able to set expectations with the customer or prospect for what will be possible to report on within StealthAUDIT, since it's limited by the target system and not STEALTHbits' software.
