Summary:

In this hotfix we fixed various issues relating to StealthDefend 2.7 SP1:

  • Fixed an issue where AD Sync was failing due to invalid credentials when processing groups.
  • Fixed an issue where the First Time Client Use threat was triggering multiple times for the same user/client combination.
  • Updated the installer to install .net version 5.0.12

Submitted by:

Greg Connors

Affected Versions:

Prior versions of 2.7 SP1
