Instructions:
The following SI modules should be patched:
* SI Windows Console
* SI Web Reporting

Steps to apply patch for SI Web Reporting:
1. Stop SIReportingService service
2. Stop SISchedulerService service
3. Extract and overwrite file from 'SIWebReports' folder to:
- SI 5.x-6.x: C:\Program Files (x86)\STEALTHbits\StealthINTERCEPT\SIReports\ReportingConsole\ClientBin\
- SI 7.0.x: C:\Program Files\STEALTHbits\StealthINTERCEPT\SIReports\ReportingConsole\ClientBin\
4. Start SIReportingService service
5. Start SISchedulerService service

Steps to apply patch for SI Windows Console:
1. Close the SI Win Console
2. Extract files from attached 'SIWinConsole' folder to the SI Win Console folder
3. Start SI Win Console.
