Summary:

  1. Unable to translate UNIX IDs in SAM 5.0
  2. Ability to search/filter Agents and Monitored host tabs

Behavior Before Fix:

  1. SAM currently does not use a Global Catalog when looking up UID information for Unix Activity.
  2. SAM Console is not able to filter agents or monitored hosts.

Behavior After Fix:

  1. SAM now has option to use LDAP, LDAP over SSL, Global Catalog, and Global Catalog over SSL when looking up UID information for Unix Activity.
  2. SAM Console is now able to filter agents or monitored hosts.
