Issue: Summary: 

  1. I-3476 SUVP 18-03 || Exchange 2016 CU7-CU8 || Applying KB4073392 results in stopping working of Exchange monitoring and lockdown policies
  2. I-3505 Exchange || Applying optional 'February 2018 Preview updates for 4.6-4.7.1 .Net results in not working SI Exchange policies and increasing CPU utilization
Before fix:
  1. Applying KB4073392 results in:
  • Exchange Monitoring policy: no new Exchange events
  • Exchange Lockdown policy: operations will not be blocked
  • Lot of errors in SI Exchange logs
 
  1. Applying recent .Net updates results in:
  •   Exchange Monitoring policy: no new Exchange events
  •   Exchange Lockdown policy: operations will not be blocked
  •  Performance: CPU utilization increased by ~ 20% on test environment
  • Lot of errors in SI Exchange logs
 
             List of influencing KBs:

KB 4074807 for Server 2012 R2
KB 4073700 for .NET Framework 4.6, 4.6.1, 4.6.2, 4.7 and 4.7.
After fix:
  • Exchange monitoring and lockdown policies work as expected
  • No errors in SI Exchange logs
