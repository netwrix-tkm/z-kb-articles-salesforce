The following message appears during a password reset/change: Unable to reset the Google apps password for this user: The remote server returned an error: (403) Forbidden.
1. Make sure that the Password Manager settings for the Google Apps and Google Apps Domain settings are set as described in the Password Manager Admin Guide on pages 30-31 (http://www.netwrix.com/download/documents/NetWrix_Password_Manager_Administrator_Guide.pdf) 2. Check that the Google Apps password (not Active Directory account password) is specified in the password field of the Password Manager Google Apps management settings.
