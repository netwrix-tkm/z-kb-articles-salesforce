Summary: . Ticket: AIC - Trustee Audit does not filter out deleted resources - Issue: Trustee Audit / Permissions reports was showing deleted resources. Generally clients don't want to see permissions for things that no longer exist. - Fix: Deleted resources (DeletedUSN not Null) are now filtered out of the report. . Ticket: AIC showing the Web Application link where it should show a Site Collection - Issue: When there's a web app that has a site collection with the same name, clicking on the Url link shown in the 'Site Collection Scan Summary' at the farm level, was directing to the web app level page instead of the site collection level. - Fix: Now clicking on the link will redirect the user to the Site Collection level page and its reports, not to the Web application one.
