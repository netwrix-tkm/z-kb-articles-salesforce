How to prevent Netwrix Password Manager from running at startup and/or remove the shortcut from the Start menu on multiple machines?
To perform these actions, modify the prm_client.msi file before deployment as follows
  1. Open the prm_client.msi file via any MSI editor, for example InstED.
  2. To prevent Netwrix Password Manager from running at startup, right-click 'StartupRegistryKey' in the Components left pane and delete it.

  1. To prevent the shortcut from appearing in the Start menu, right-click 'StartMenuShortcuts' in the Components left pane and delete it.
  1. Go to File > Save as and save the changes.
  2. Use the transformed prm_client.msi file for deployment.
