Summary: NAS logs don't roll over at midnight. This job restarts the logging service to force the rollover. 

Issue: Logging service doesn't roll over when it should

Instructions: Drop in JOB_LastLogon job found at \\sbitsinc.com\public\Escalation\KB Attachments and schedule at midnight against the local host (or remote FPolicy server or CEPP endpoint not supported in this job). Use a connection profile. Has error handling and output columns that describe the result of each step.

Error codes for start and stop options:
 
Stop Service - https://msdn.microsoft.com/en-us/library/windows/desktop/aa393673(v=vs.85).aspx
Start Service - https://msdn.microsoft.com/en-us/library/windows/desktop/aa393660(v=vs.85).aspx
