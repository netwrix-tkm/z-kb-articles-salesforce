Issue: Summary:
This Hotfix addresses a number of high priority issues around assigned and executing threat response action groups as well as issues with with evaluation of console access for AD Groups.
 
Behavior before Fix:

  • Threat Configuration Page fails to properly set Threat Response options
  • Automated Threat Response on Threat detection would NOT RUN.
  • AD Groups with assigned privileges in console access will not be properly assigned access.
 
Behavior after Fix:
Threat Response on Threat Configuration page assigns action groups to threats properly. Action Groups will be executed at time of threat detection properly. AD Users that are a member of an AD Group scoped with Console Access will inherit rights properly.

2. Double click the installer to launch the upgrade process
