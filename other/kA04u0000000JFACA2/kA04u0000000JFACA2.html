Summary: Summary: This hotfix prevents an error from appearing when using angle brackets in Report names. Issue/Request: When using angle brackets in Report names, an "illegal characters in path" error would appear. Fix/Enhancement: Angle brackets are acceptable characters to use in Report names; no error appears.

1. Unblock the hotfix zip file in the windows property dialog if an unblock button exists there.

3. Exit StealthAUDIT console
