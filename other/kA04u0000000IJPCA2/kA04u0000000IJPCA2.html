Instructions:

  1. Unblock the hotfix zip file in the Windows property dialog, if an unblock button exists there
  2. Close all instances of StealthAUDIT (check task manager under processes for all users)
  3. Exit StealthAUDIT console
  4. Replace the provided files in the 'Private Assemblies' folder
  5. Run the FixChanges.exe tool
  6. Restart StealthAUDIT.

 

* Make sure to run the Fixchanges.exe tool described in the KB Lost Settings After Upgrade to SA 10.0 as part of this hotfix.
