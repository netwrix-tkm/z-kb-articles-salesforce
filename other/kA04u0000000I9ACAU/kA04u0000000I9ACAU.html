Summary: How to enable TLS / SSL in StealthRECOVER

Issue: See instructions to enable TLS/SSL in RECOVER

Instructions:    1. Obtain a Web Cert.
    2. Install the Cert into the Personal location in the Local Computer store.
    3. Copy the Cert into the Trusted Root location in the Local Computer store.
    4. Open the appsettings.json file in the StealthRECOVER installation directory.
    5. Change the value of "Certificate" from "" to the value of the Cert's Subject field.
    6. Restart the StealthRECOVER service.

NOTE: After making these changes, you must use both HTTPS and the FQDN to connect to the web portal (e.g., https://recover01.domain.com:5000, not http://localhost:5000, https://localhost:5000, or http://recover01.domain.com:5000.)

There's a trick to it because we only look at the certs in the personal store and those aren't necessarily trusted so these instructions account for that
