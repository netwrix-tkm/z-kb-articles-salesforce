Issue: Summary: EM won't run after 5.2 upgrade - could not read configuration
Behavior before Fix:
The root cause of this problem is IP Addresses collection which contains just one item which has NULL value. This cause EM to fail to initialize. Also WinConsole crashed on attempt to open of edit this collection.
Behavior after Fix:
EM code updated to handle such cases and cleanup such miss-configured collections. Also WinConsole was modified to handle this case on opening such collections and on attempt to modify it.
