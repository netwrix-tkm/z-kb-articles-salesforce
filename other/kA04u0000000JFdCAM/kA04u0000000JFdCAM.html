Behavior Before Fix:
database performance issue which causes a slowdown. This is because there is a missing index on the Resources table which causes inserts of new resources to take longer and longer as the scan goes on and TLS Connections are failing on subsequent web requests.


Behavior After Fix:
ServicePoint Flag added to CSOM query and necessary indexes added to Resources Table.
