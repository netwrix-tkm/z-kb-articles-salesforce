Install SbPAM Browser Extension Hotfix

1. Unzip the sbpam-browser-extension-[version].zip file to your hard-drive.

2. Install using developer mode.

3. In Chrome, click the icon that looks like a puzzle piece, then click "Manage Extensions"

4. On the Manage Extensions page, turn on 'Developer Mode'

5. Then click 'Load unpacked' and select the folder that you unzipped the distribution to

6. The browser extension will now be installed. Version "0.0.1" refers to an unreleased build. Ignore the 'Errors' button.

7. Click the puzzle piece icon on the toolbar again, then click the pin to display the extension hotfix icon on the toolbar instead of the release version. Pin the gold icon instead of blue icon.

8. Open the Gold extension to test the developer build

Uninstall SbPAM Browser Extension

1. In Chrome, click the icon that looks like a puzzle piece, then click "Manage Extensions"

2. To revert to the released version you just need to pin the blue icon and unpin the gold

3. Remove the gold SbPAM extension.
