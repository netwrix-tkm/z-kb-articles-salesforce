Summary: StealthINTERCEPT Resource Leak

Issue: Title:  SIWindowsAgent_5.0.0.797_5.1.0.428__5.2.0.193_SI-3610
Summary:
StealthINTERCEPT Resource Leak
Behavior before Fix:
Problem Description:
In some cases it is possible that AD Monitor will try to allocate big amount of memory which will be rejected by OS and after this situation AD Monitor can go to infinite loop.
Behavior after Fix:
Element in the structure which represents attribute value size changed from 64bit number to 32bit. Also added code to avoid infinite loops in this place even in the case when AD Monitor was unable to allocate needed memory.
