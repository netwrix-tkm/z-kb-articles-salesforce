Issue:
When using the Exchange SDD SS with Exchange Online after running only Azure Active Directory Inventory the EX_SDDResults_MailboxSummary table was blank.  This fix will make it so it works when Active Directory Inventory has not been run but Azure Active Directory Inventory has been run. 
 

Instructions: 1. Close SA
2. Replace the DataAnalysisTasks.XML file with the one included in these 2 locations:
\StealthAUDIT\InstantSolutions\GROUP_Exchange\GROUP_7. Sensitive Data\JOB_EX_SDDResults
\StealthAUDIT\Jobs\GROUP_Exchange\GROUP_7. Sensitive Data\JOB_EX_SDDResults
