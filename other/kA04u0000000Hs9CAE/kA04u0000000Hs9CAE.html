Summary:

After install of SAM agent 6.0.690 or a few days into running "Agent services may have recently crashed" message is seen

Behavior Before Fix:

Windows monitoring service crashed when invalid INI  sections contained no outputs or destinations.

Behavior After Fix:

SAM agent now removes invalid INI  sections without any outputs or destinations, preventing windows monitoring service crash.
