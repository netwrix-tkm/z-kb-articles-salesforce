Summary: Ticket: AIC - Remove open access remediation rights updates Issue: When performing an open share remediation through the AIC console, the SA_FSAA_Rights table was being updated to reflect the high risk trustees' rights removal, this unintendedly set the SA_FSAA_Hosts table's GUID and USN to 0, which prevented subsequent FSAA scans to be correctly bulk imported into the database. Fix: All code that updates Rights and Host tables was removed from AIC. Share remediation still works.
