Summary:  FSAM crash when attempting to copy/move paths of 260+ characters with Preserve file access option on

Before Fix: Unhandled exception would occur when attempting to copy/move paths of 260+ characters with Preserve file access option enabled. FSAM applet would crash unexpectedly.

After Fix: No crash when copying/moving paths of 260+ characters with preserve file access option enabled.
