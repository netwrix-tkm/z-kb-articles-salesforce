Summary: By default, the AIC only sends emails to the primary owner of resources. This will show you how to enable email notifications for all users.

Issue:

The AIC by default will only send email notifications to the assigned primary owner of resources. This can be changed by editing a config file.

No IIS reset is required.

Instructions:

Open the C:\inetpub\wwwroot\StealthAudit Compliance\Bin\common.config file with notepad and set the "NotifyAllOwners" parameter to 'True' and navigate to File > Save.

 

Result: All owners assigned to resources should now receive email notifications concerning their resources.
