Instructions: The 'hardeningEnabled' parameter should be used to automatic hardening of SI Agent after installation.
Example of command line to install SI Agent with automatic hardening:
"SI Agent.exe" /q ENTMGR_IPADDRESS=20.0.0.1 FILE_MONITOR_INSTALL=TRUE AD_MONITOR_INSTALL=TRUE FILE_MONITOR_INSTALL=TRUE SAFEMODE=FALSE ADDFWRULES=FALSE CONFIGPARAMS="hardeningEnabled=TRUE"


Patches were not prepared and posted because the fix is in the Agent Installer.
Therefore SI Agent installers  should be used to install SI Agents with automatic hardening.
