Summary:   Arithmetic overflow Running FS Forensic Jobs

Before Fix: SQL analysis was casting folderID values to varchar (4) which would cause an Arithmetic overflow error for any FolderID with a value of 10000 or greater.

After Fix: SQL analysis updated to cast folderID values to varchar (max) to avoid any Arithmetic overflow issues.
