Summary: Enhancement to allow ADI DC to collect attributes as an extended attribute even if we collect the attribute already.

Issue: Summary:
Enhancement to allow ADI DC to collect attributes as an extended attribute even if we collect the attribute already.
 
Behavior Before Fix:
When attempting to add an extended attribute that we already collect, the wizard gives the user a popup of "ADInventory collects the attribute '[attribute]' of the [user/group/computer] by default.  Please choose another attribute to be collected.".  User is *unable* to add the attribute.
 
Behavior After Fix:
When attempting to add an extended attribute that we already collect, the wizard gives the user a popup of "ADInventory collects the attribute '[attribute]' by default for the following classes: [user/group/computer]  Would you still like to add this attribute?"  The user is *able* to add the attribute.
