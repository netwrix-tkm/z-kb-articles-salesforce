Summary: Instructions on how to add additional security groups to the AD_SensitiveSecurityGroups job/report.

Issue: Customer would like to add additional security groups to the sensitive security group report. By default, this report includes Enterprise Admins, Domain Admins, Schema Admins, DNS Admins.

Instructions:

  1. Navigate to ".Active Directory Inventory > 1_AD_Scan > Results > ADInventory_GroupsView"
  2. Click the magnify glass in the upper right hand corner and select "SamAccountName"


 

 
  

 

  1. Enter the group name that you want to add to the Sensitive Security Group and hit "Enter"

 

 

  1. Scroll over until you find "ObjectSid" and copy it to the clipboard.

 
  

 

  1. Navigate to Active Directory > 1.Groups > AD_SensitiveSecurityGroups
  2. Expand AD_SensitiveSecurityGroups job
  3. Expand Configure and click Analysis
  4. Open the "Calculate Effective Membership" analysis and click "Configure Analysis".
  5.  
      

    On Line 12, you will find the following:

ORIGINAL: INSERT INTO #SIDs (Value) VALUES ('S-1-5-21%-512'), ('S-1-5-21%-519'), ('S-1-5-21%-518')
ADD: Add/paste the additional SID to the end up this line in the following format ",('SID')"
AS AN EXAMPLE: , ('S-1-5-21-189865919-1311902141-2254347052-1143')
FINAL RESULT: INSERT INTO #SIDs (Value) VALUES ('S-1-5-21%-512'), ('S-1-5-21%-519'), ('S-1-5-21%-518'), ('S-1-5-21-189865919-1311902141-2254347052-1143')
 

  1. Once complete, click "Save and Close" and click "Save".
  2. Rerun the AD_SensitiveSecurityGroups job and the new group should appear in the report.
