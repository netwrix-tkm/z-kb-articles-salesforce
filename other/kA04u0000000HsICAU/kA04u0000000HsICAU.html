Summary:

SAM agent 6.0.0.798 upgrade has caused service to restart constantly and is blue screening devices

Behavior Before Fix:

When SBTService enumerates Mount Points for a Volume, there was a bug that caused it to get stuck if it fails to retrieve the list of Mount Points. If one of them fails, then it gets stuck.

Behavior After Fix:

Implemented fix for SBTService getting stuck (caused by mount points).  And also added agent option to enable driver ETW logging.
