Summary: This is a one off Hotfix for Sandridge. The customer was unable to bulk import because they had Collaborations for Folders that were not included in the Folders table. This led to unexpected null values that causing imports to fail with the message "Specified Cast is not Valid". Dev and QA were unable to reproduce their issue in house, but this fix will allow the customer to successfully import.
