Issue: Summary:
 
32039 - PS DC: Weak Password Job Fails to use correct credentials
Run SA_AD_WeakPasswords with a Domain Admin against multiple domains with a connection profile that has credentials for both
Job will successfully work against the domain StealthAUDIT is in, but fail against the foreign domain(s) with a null-valued expression error. This error is due to a LDAP query we're running not populating because we're not using the correct credentials.
