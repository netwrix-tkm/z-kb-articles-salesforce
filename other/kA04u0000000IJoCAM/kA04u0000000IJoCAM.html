Summary: ServiceNow Error When inserting value in the Caller field

Issue: Summary:
ServiceNow Error When inserting value in the Caller field
 
Behavior Before Fix:
Entering in information into some of the fields on the new incident page of the ServiceNow action module wizard caused unhelpful errors like "The remote name could not be resolved."  Some issues existed in the wizard that caused the data entered into some of the fields to not be used when creating incidents.
 
Behavior After Fix:
Lookup functionality was added to the New Incident page in order to prevent incorrect data from being entered.  The error messages should now be helpful.  Data entered on the New Incident page is now being respected when incidents are created.
