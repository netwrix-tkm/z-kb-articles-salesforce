Summary:

SI-4455 60483 - EPE is not allowing password over 127 characters (this needs to be increased to new MS limit of 256)

Behavior before Fix:

EPE policy can't operate with new Microsoft password length limit: 256 symbols

Behavior after Fix:

Increased limit for the password length from 127 to 256

Changed the upper limit for UI filters of SI EPE policy

Result: EPE policy can operate with new Microsoft password length limit: 256 symbols
