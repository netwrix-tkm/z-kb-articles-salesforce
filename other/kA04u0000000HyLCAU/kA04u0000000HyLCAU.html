Instructions:

  1. Unblock the hotfix zip file in the Windows property dialog, if an unblock button exists there
  2. Close all instances of StealthAUDIT (check task manager under processes for all users)
  3. Exit StealthAUDIT console
  4. Stop the StealthAUDIT Web Server service
  5. Replace the provided files in the appropriate directory
  6. Reconfigure the WebServer.exe.config file (alternatively, keep a copy of the original one to replace the new file)
  7. Start the StealthAUDIT Web Server service
  8. Start StealthAUDIT / SA Web Server.
