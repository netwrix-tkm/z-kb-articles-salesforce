Issue: Summary:  Edit of existing FS policy with "Enable Automatic Lock-down" can't be saved
Before fix: Because of error in logic which shows if File System policy has Auto blocking configured it was impossible to save such policy because it required to have at least one path selected in the Blocking registration of this auto blocking policy.
After fix:  Logic was fixed to correctly indicate if policy has auto blocking configured.
