I'm wondering how I might be able to create an install for Netwrix Password Manager client (Logon Prompt Extension) with no user interaction so that I can push it out through a GPO.
For installation of the client via Group Policy you can use a separate prm_client.msi installer. It can be found in the program installation directory after installing a server part (by default C:Program Files (x86)NetWrix Password Manager) GP installation is described in the procedure 4 page 11 of the Administrator's guide
