SA_9.0_094

Summary:  This hotfix contains an enhancement to the SQL Solution Set.  The Sysadmins reports now properly expand AD group membership.

Before Fix:  The following situations were not handled when calculating the SQL permissions:

  • AD group membership was not being calculated when control server permissions was directly applied to an AD group, only when inherited through a SQL role.
  • Sysadmin role was not being calculated when directly applied to AD groups where it was inherited or directly applied
  • Control Server membership path calculation was always calculated creating extra SQL work.

After fix:

  • AD group membership should now be correctly calculated throughout the SQL solution set
  • Control Server membership path calculation can be controlled by a new parameter in the analysis task
