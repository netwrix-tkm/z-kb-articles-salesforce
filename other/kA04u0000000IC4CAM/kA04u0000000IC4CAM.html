Description:

This hotfix includes a LARGE number of fixes and enhancements for the Honey Token functionality of StealthDEFEND as well as providing a variety of fixes for the threats and investigations.

Notable Enhancements

  • Honey Token Names are now defined statically instead of using dynamic character variables
  • Honey Token deployment script utilizes a new “fallback” mode that allows the honey token to be deployed even if the StealthDEFEND server is unreachable
  • Honey Token Domain is now configurable

 

Submitted by:

Lee Berg

Affected Versions:

Prior versions of 2.5
