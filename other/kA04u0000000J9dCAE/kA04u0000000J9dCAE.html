In order to use this fix, add the following XML config in the FSAA_SETTINGS section of the job xml in both the DFS Scan and DFS bulk import job: True
 

Instructions:

  1. Close all instances of StealthAUDIT.
  2. Update all DC files in your %SAInstallDir%DC folder
  3. Update all PrivateAssemblies in your %SAInstallDir%PrivateAssemblies folder.
  4. Launch StealthAUDIT. Run the 'Update Proxy Service' query under the Maintenance tab in the FSAA data collector against your proxy servers (if running proxy service).
