Summary:

SI Credential Provider Prevents Auto-Fill of Logged On User

Behavior before Fix:

Logged On User is pre-populated despite of Enabled “Interactive logon:Dont display last signed-in” GPO setting

Behavior after Fix:

Do not show a cached user name after Reboot or Power on

Logoff/Logon – show cached user name unless enabling of "Interactive logon:Dont display last signed-in" GPO setting
