Summary: Sensitive Data Scans return that a file matching $~thisdoc.docx is not supported

Issue: Sensitive Data scans or StealthSEEK returns an error message saying that a known supported file type is not supported. If you look closely all these file types begin with a $~ and are backup/temp files.

Instructions: Backup/temp files are not supported in Sensitive Data Discovery/StealthSEEK.  While the file extensions are supported the type of file is not.  Please see the Supported File formats on the documentation share for what file types are supported.
