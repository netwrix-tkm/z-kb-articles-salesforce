SA_11.6_001

Summary:   AD_WeakPasswords - Could not load file or assembly 'BouncyCastle.Crypto –
Prior to 11.6 AD_WeakPasswords did not use 'BouncyCastle.Crypto and it was only laid down with a Box Data Collector License.  This HF is to provides a work around if the next CU (Dec 1, 2023) cannot be used.

Instructions:
  1. Unblock the hotfix zip file in the windows property dialog, if an unblock button exists there. 
  2. Close all instances of StealthAUDIT (check task manager under processes for all users)
  3. Update all PrivateAssemblies in your %SAInstallDir%PrivateAssemblies folder.
