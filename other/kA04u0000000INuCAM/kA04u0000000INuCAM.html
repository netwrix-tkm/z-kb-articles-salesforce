Issue: Summary:
SMPC-5000: Fail to login to AIC when additional domain added

  • Issue/Current behavior: AIC attempts to enumerate trusted domains when starting up, for the client, adding a new domain will result in it not being resolved.
  • Fix/Enhancement: Now there will be no attempt to enumerate all trusted domains when the parameter "ADServiceEnumeratedDomains" has values set, only those values will be resolved.
