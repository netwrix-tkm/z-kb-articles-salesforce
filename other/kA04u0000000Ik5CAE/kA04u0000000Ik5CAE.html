Summary: Fixed a bug where hosts specified by FQDN we being scanned using the shortnames.
Issue/Request: In environments where an FQDN is required to access host data, reverting back to a shortname prevents the scan.
Fix/Enhancement: We now prevent SA from switching between FQDN and shortname, and will use whichever is specified in the Host List.
