Summary: SI-4001 40807 - SI collection - unable to remove objects

Issue: Summary: SI-4001 40807 - SI collection - unable to remove objects
Behavior before Fix:
There is is subset of collections which has only single column for values and presented in the UI using ListBox control: "Domains & Servers", "Contexts", "Classes", and"Attributes".
This control was configured to show data sorted, but data in associated collection was sorted according to the order how data was added to the collection. And since deletion of the data in the collection is implemented by position number in the ListBox therefore if data was added to the collection in not alphabetical order then it can delete wrong item.
Behavior after Fix:
Now ListBox is reconfigured to display data in the same order as it present in the associated collects - i.e. in the order of how it was added to the collection.
