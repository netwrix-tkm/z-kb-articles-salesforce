Summary: Duplicate mailbox aliases

Issue: When running EX_Perms job against an Exchange server that has multiple mailboxes with the same alias an Error is seen.  No data is collected for the users that have the same Exchange Mailbox Alias.
 

Instructions: 1.            Unblock the hotfix zip file in the windows property dialog, if an unblock button exists there.
2.            Close all instances of StealthAUDIT.
3.            Update all PrivateAssemblies files in your %SAInstallDir%PrivateAssemblies directory
4.            Update all DC files in your %SAInstallDir%DC directory
