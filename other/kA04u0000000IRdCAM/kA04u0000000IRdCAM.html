Issue: Summary:
Feature Request - Browser compatibility message in AIC

  • Issue: Silverlight is no longer supported in many internet browsers. This leads to issues when non-technical staff access AIC to perform entitlement reviews. Chrome and Firefox will continually show a message asking to install Silverlight.
  • Fix: A message is now displayed in those browsers to let users know that they are not supported: "The Access Information Center needs Internet Explorer with Silverlight. Navigate from Internet Explorer to continue."
