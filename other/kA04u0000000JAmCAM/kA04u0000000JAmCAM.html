Summary: Add configurable option to determine whether to zip emailed report files. As part of an earlier enhancement to 7.1 a decision was made to automatically zip our report files that are set to be emailed by default. This was later changed to accommodate customer security requirements; as many customers are not able to send/receive zipped files as an email attachment. This hotfix introduces the zipping functionality as an optional checkbox, that includes the ability to set a threshold file size for when a report should be sent as a zipped file.
