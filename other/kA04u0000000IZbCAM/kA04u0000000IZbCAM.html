Issue: Summary:  send syslog issues
Before fix:

  • SIEM event consumer (and all others) was active only when the checkbox "Send to Events DB" is checked on the actions policy page.
  • Events queue was limited with 500000 entries.
After fix:
  • Every event consumer works independently from others.
  • Removed the events queue limit.
