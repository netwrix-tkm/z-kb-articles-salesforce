SI missing lockout events
Behavior before Fix:
There a logic in ADMonitor to filter out TGT events which don't have enough information as noise events. This also includes the case when client computer name is empty. Therefore, these particular authentication events were filtered out.
Behavior after Fix:
Now logic is modified in the way to keep events which have client IP address instead of client machine name
