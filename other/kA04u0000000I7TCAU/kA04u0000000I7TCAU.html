Summary:   SP SS - Effective access reports for SharePoint online sites not showing all users

Before Fix: Azure AD Users inside Azure AD Groups that were nested inside SharePoint Groups were not showing to have effective access.

After Fix: Update SA_SPAA_GetTrusteeMembership function to accurately calculate effective access for Azure AD users.
