Summary: SI Investigate tab bug in search
Behavior before Fix:
Before fix:
Sysmbols of percent and understrike are wildcard symbols for LIKE statement of SQL. Understrike - any one symbol, Percent - any symbols.
This symbols in filters of investigate page cause to show excess results. Example: "John_Doe" filter will return "John_Doe", "John.Doe", "John Doe" etc
Behavior after Fix:
This symbols works as plain symbols. Example: "John_Doe" filter will return only "John_Doe" results.
