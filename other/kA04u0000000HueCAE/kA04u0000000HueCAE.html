Summary:

SMPC-12080 \ Escalations 60048:  Error when running a SQLViewCreation with CSV export

Issue/Request: When running a SQLViewCreation analysis with CSV export enabled, customer was getting an error related to the DotNetZip.dll assembly where the expected version was 1.10.1.0, while the customer had 1.15.0.0.

Fix/Enhancement: The SQLViewCreation analysis module was rebuilt so it uses DotNetZip.dll V. 1.15.0.0.
