Summary:

  1. Improve AD resolution & caching
  2. SUVP 21-10 | 2019 Server Only | No NTLM events with KB5006672, release date: 10/12/2021

Behavior Before Fix:

  1. No AD timeouts. SAM does not impose its own timeout on AD lookups; SAM uses the default system resolver.
  2. NTLM events are not coming due to signature change in NlpLogonSamLogon

Behavior After Fix:

  1. A customizable timeout for AD lookups in the agent settings, so a customer may set a cap on the maximum time. When the time-out interval elapses, the lookup fails. The resolution process continues in background. Eventually it will complete and populate the cache; SAM allows you to specify DCs explicitly. If they are not available, fall back to the system resolver.
  2. Signatures updated; NTLM must be fully functional.
