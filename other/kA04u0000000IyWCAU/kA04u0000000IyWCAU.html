Summary: How to resolve blank published reports when vault service is enabled

Issue: Prior to HotFix 8.2.052, the reports section of the published reports would show up blank after enabling the Vault service.  Switching back to application would resolve the issue.  No errors appear in the web logs.

Instructions: To resolve apply the following hotfix https://downloads.stealthbits.com/access/files/SAHotFixes/8.2/SA_8.2_052.zip ensuring that when setting up the Web Service a domain account with StealthAUDIT administrator role is used to run the service.  This is a change from default running service as System account.
