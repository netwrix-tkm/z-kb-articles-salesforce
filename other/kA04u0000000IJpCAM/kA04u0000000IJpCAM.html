Summary: GPO Monitoring does not work.

Issue: Summary:
GPO Monitoring does not work.
Behavior before Fix:
SBTService is responsible for sending notifications to GPO Differencing Service about changes in GPO related files. The problem was that the file path was presented in the format like '\\?\Volume{GUID}\...' which GPO Differencing Service can't understand.
Behavior after Fix:
Now file path format changed to 'C:\Windows\...' which GPO Differencing Service can understand.
