Summary: Enhancement - Order scheduled actions table on target by default. Previously, the scheduled actions table would sort on the full string value of the path. This change now has it so the default view of the scheduled actions table is ordered by target on default. This allows you to easily find scheduled tasks as scheduled jobs/groups are in the same order as would be found in the jobs tree.
