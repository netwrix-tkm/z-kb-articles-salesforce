Summary: Problems with Exchange on SB environment

Issue: Summary:
Problems with Exchange on SB environment
Behavior before Fix:

  • Query delegate info on storage open. This causes significant delay.
  • Some hooks perform more activity even if it is not needed according to active rules
Behavior after Fix:
  • Stop query delegate info during storage open. Query moved to set and delete operations and only if folder permission blocking is enabled.
  • Don't check if message can be really opened. This can cause some message events with unknown attributes.
  • Make several hooks lighter in case when some operations are not needed for active policies.
