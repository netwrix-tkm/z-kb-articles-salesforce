Summary: Client has NFS shares presented on Netapp 7 mode using Windows authentication. How can we include the user map from the appliance to show the permissions correctly in AIC?

Issue: We have worked through this issue with Honeywell using the attached job. This job is responsible for:

  • reading /etc/usermap.cfg file to parse AD-> Unix Mapping
  • reading /etc/passwd file to parse Unix users
  • correlating the two to provide view which is then used by AIC

This job is specific to NetApp machines, where AD users are being mapped to unix accounts through the usermap.cfg file.

This job will require enough privileges on the netapp to run the rdfile command against the files listed in the bulleted list above

Instructions: Use job located here:
