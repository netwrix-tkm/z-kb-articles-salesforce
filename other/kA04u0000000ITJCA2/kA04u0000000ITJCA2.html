Summary:  SPAA SDD scans have hard-coded timeout of 60 minutes

Before Fix: SPAA SDD scans will timeout and completely stop if a file was in queue for 60 minutes.

After fix: Removed that timeout from SPAA code. Each individual file processed will now go thru the SDD File scanning timeout parameter, same functionality for FS SDD scans.
