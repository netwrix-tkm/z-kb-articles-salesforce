Error when trying to view ADInventory_ExtendedAttributesPivotView: FOR XML could not serialize the data for node 'NoName' because it contains a character (0x0007) which is not allowed in XML. To retrieve this data using FOR XML, convert it to binary, varbinary or image data type and use the BINARY BASE64 directive.
