Summary:

2019 Server only | LSASS may crash with KB5004244, released 07/13/2021

Behavior Before Fix:

HandleTGSRequest was resolved by Windows 2k16-specific signature of HandleTGSRequest which is the signature of different function on 2k19. As the result other function was hooked instead of HandleTGSRequest.

Behavior After Fix:

Windows 2k16 uses 2k16-specific signatures & 2k19 - 2k19-specific signatures of HandleTGSRequest. Additional 2k19 signature was produced for HandleTGSRequest.
