Summary:  SQL DC Permissions imports into the AIC was removing other data from custom imports.  This fix allows custom import data in the SA_AIC_FlexibleLocalGroupMembership and the SA_AIC_FlexibleHosts to remain after the permissions analysis is run.

  • *important* when importing data into the SA_AIC_FlexibleHosts table, the ScanType must be unique for the data to not be overwritten.

Before Fix:  Custom import data in the FlexibleHosts and FlexibleLocalGroupMembership was overwritten by the SQL DC permissions analysis.

After fix:  Custom import data is no longer overwritten.
